Welcome To My Page!

This is my BIG website for IT 161. I have provided resources that will teach you about Accessibility, Social Media Optimization, Web Forms, Search Engine Optimizations, and Galleries, as well as tips for improving your skills in the field of web development.

In today's tough competitive world, in order to successfully interview for a web job, contribute to a web team, and provide guidance for web clients, you should familiarize yourself with current and coming web topics and coding practices. Here are a couple of tips that will help you get started:

Get Your Basics Down

We all understand the importance of real world learning, but one should never under estimate book learning. Brush up on your theory skills by starting with the basics like The Non-Designers Design Book or maybe going for something more specific like WordPress Web Design for Dummies.

Join the Open Source Community

When you join an open source community, not only are you able to work with a variety of developers, but you have access to some incredible coding as well. Whenever you are feeling overwhelmed on a project, or need assistance, you can generally reach out to the community for assistance.

Create Your Own Sandbox

In order to improve our design skills, we need to do more than just following tips, we need to practice. To do this, make yourself a safe space using a localhost, where you can try out new design principles without possibly breaking anything. There are guides for creating local hosts for Windows and on Mac.
